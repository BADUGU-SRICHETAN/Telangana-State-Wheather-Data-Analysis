# Telangana-Wheather-Data-Analysis
 Interactive Power BI dashboard for analyzing Telangana weather data, including rainfall, temperature, humidity, and wind speed across districts. The project provides insights into weather trends, seasonal variations, and district-wise climate patterns using interactive visualizations and filters.


# Project Description
This project analyzes weather patterns across different districts using Power BI dashboards. The dashboard provides insights into temperature, humidity, rainfall, and wind speed trends using interactive visualizations.

# Business Problem
The objective of this project is to analyze weather conditions across multiple districts and identify climate trends that can support environmental monitoring and data-driven decision-making.

# Features
- Temperature Analysis
- Humidity Analysis
- Rainfall Analysis
- Wind Speed Analysis
- Interactive Filters and Slicers
- KPI Cards and Gauge Charts

# Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset
